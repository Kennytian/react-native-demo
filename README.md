# React Native Demo

## Installation
1. `git clone https://github.com/Kennytian/react-native-demo.git`
2. `cd react-native-demo & yarn install`
3. `react-native run-ios`

## Dependencies

- prop-types
- react
- react-native
- react-navigation
- eslint
- pre-commit
- jest
